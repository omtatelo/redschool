Nosotros

Y bien, aquí empieza la aventura de Redschool. Estamos muy emocionados de iniciar lo que creemos será la herramienta para búsquedas más completa y especializada a la hora de escoger en dónde estudiar o en dónde inscribir a tus hijos. Como esto es el principio iremos incorporando poco a poco cada una de las herramientas y opciones educativas de modo que pronto las tengamos todas.

Redschool se conforma de un equipo de mercadeo, diseño y desarrollo web que están trabajando de manera incansable para crear una base de datos en la cual las escuelas puedan inscribirse. Ahora mismo puedes ver un ejemplo de cómo quedarán las páginas completas aquí así que si eres dueño de una institución educativa, anímate a sumarte a esta iniciativa y contáctanos.
